About this Project

This project explores how key musical features like energy, danceability, speechiness, and loudness vary between highly popular and less popular songs across genres and over time. Using two curated datasets from Spotify's API, this project visualizes musical trends, genre characteristics, and evolution patterns to better understand what drives popularity in music today.
